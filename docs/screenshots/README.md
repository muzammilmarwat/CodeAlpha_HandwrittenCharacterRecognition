# Final App Screenshots

These screenshots were captured from the local Streamlit app using non-personal MNIST sample and canvas data.

Run the app locally with:

```bash
streamlit run app/app.py
```

| File | Status | Shows |
| --- | --- | --- |
| `home.png` | Available | Compact hero, app title, sidebar, input source options, and model/test accuracy badges |
| `canvas_input.png` | Available | Draw Digit mode, visible handwritten digit, Clear canvas control, and Recognize Digit button |
| `prediction_result.png` | Available | Original image, processed 28x28 preview, saliency heatmap, saliency overlay, predicted digit, confidence, confidence band, and probabilities |
| `model_information.png` | Available | Model metrics, metadata, preprocessing summary, and evaluation visuals |
| `error_analysis.png` | Available | Error metrics, common confusion pairs, per-class accuracy, and analysis table |
| `about_project.png` | Available | Project overview, dataset, workflow, technology stack, limitations, and download center |
