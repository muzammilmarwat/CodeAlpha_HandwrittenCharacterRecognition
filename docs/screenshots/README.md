# Screenshot Plan

Capture these screenshots after Phase 4B visual verification:

- `home.png`: App hero, sidebar navigation, and prediction workspace entry state.
- `prediction.png`: Upload or sample input selected with preprocessing guidance visible.
- `result.png`: Completed prediction showing result card, top-3 probabilities, confidence visual, and saliency.
- `model_information.png`: Model metrics, metadata, training curves, confusion matrix, and report downloads.
- `error_analysis.png`: Error metrics, confusion-pair chart, per-class performance, and analysis images.
- `about.png`: Project overview, workflow, tech stack, limitations, and educational disclaimer.

Screenshots should be taken with the app running from:

```bash
streamlit run app/app.py
```
